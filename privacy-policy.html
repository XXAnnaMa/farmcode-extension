🌾 FarmCode Privacy Policy

Last updated: April 28, 2026

Data Collection

FarmCode does not collect, store, or transmit any personal data to external servers. All data is stored locally in your browser using chrome.storage.local.

API Key

Your Anthropic API key is stored locally in your browser and is only used to make direct requests to the Anthropic API. It is never shared with or transmitted to FarmCode's servers (which do not exist).

LeetCode Data

FarmCode monitors LeetCode submission results via DOM observation to detect Accepted submissions. No submission content or personal LeetCode data is stored or transmitted externally.

Contact

For questions, please open an issue on GitHub.
